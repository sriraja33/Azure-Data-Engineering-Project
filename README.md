# Azure-Data-Engineering-Project
## Introduction

Designed and implemented an end-to-end data engineering solution by extracting data from GitHub APIs, ingesting it into an Azure Data Lake using Azure Data Factory, transforming and curating data with Azure Databricks, and enabling analytics through Azure Synapse Analytics with interactive dashboards built in Power BI.
